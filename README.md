# Usage
1. Set up Twitter App
    * Set callback url http://localhost:8080
1. Download binary
    * https://github.com/SpringMT/twitter-token/releases
1. Start twitter-token
1. Enter Consumer Key and Consumer Secret
1. Access http://localhost:8080
